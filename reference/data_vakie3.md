# Population data by municipality

Population development and projections for Finnish municipalities,
covering observed values (2000–2019) and projections (2020–2040).

## Usage

``` r
data_vakie3
```

## Format

A data frame with the following columns:

- tunnus:

  Municipality code

- nimi:

  Municipality name

- Maakunta:

  Province

- value:

  Population count

## Source

Statistics Finland
