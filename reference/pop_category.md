# Population development by municipality category

Components of population change by urban-rural municipality category.

## Usage

``` r
pop_category
```

## Format

A data frame in long format with 360 rows and 4 variables:

- year:

  Year

- category:

  Urban–rural municipality category

- value:

  Population change in persons

- variable:

  Demographic component of population change

## Source

Statistics Finland. *Migration and population change database*.
<https://stat.fi/fi/palvelut/tilastodatapalvelut/tilastotietokannat/vaestotilastopalvelu/muuttoliike-ja-vaestonmuutokset>
