# Human Development Index (HDI) data

A dataset containing Human Development Index (HDI) indicators published
by the United Nations Development Programme (UNDP). The data include
country-level HDI values and related indicators for multiple years.

## Usage

``` r
data(HDI_data)
```

## Format

A data frame with one row per country-year and the following variables:

- Country:

  Country name

- HDI:

  Human Development Index value

## Source

United Nations Development Programme (UNDP). *Human Development Report
2025, Statistical Annex*. <https://hdr.undp.org/>
